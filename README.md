# EXELIO Landing Page

Static English landing page draft for EXELIO.

`The Cafe Where Only Monsters Come as Customers` is treated as external story
content: a standalone spin-off / post-Early-Access-version short fiction series
set in the EXELIO world. The page title, primary subject, and Steam CTA remain
centered on EXELIO.

- `index.html`
- `styles.css`
- `script.js`

## GitHub Pages Deployment for `alfrog-exelio`

1. In GitHub, open the `alfrog-exelio` organization.
2. Create a new public repository named `exelio-lp`.
3. Upload the files in this folder to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `robots.txt`
   - `README.md`
   - `assets/`
4. Open the repository settings.
5. Go to `Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select:
   - Branch: `main`
   - Folder: `/root`
8. Save, then wait for GitHub Pages to publish the generated URL.

Expected URL:

```text
https://alfrog-exelio.github.io/exelio-lp/
```
